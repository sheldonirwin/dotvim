Vim Configuration
=================
by Sheldon Irwin

Installation
------------

Clone the repository:
	
	git clone git://github.com/sheldonirwin/dotvim.git ~/.vim
	
Create Symbolic Links:
	
	ln -s ~/.vim/vimrc ~/.vimrc
	ln -s ~/.vim/gitignore ~/.gitconfig
	ln -s ~/.vim/gitignore ~/.gitignore
